Everything We Built

A complete sanitation management system engineered from the ground up for Ferrero's ISO compliance requirements.

📋 Core

ISO Sheet Management

Weekly cleaning checklists structured to ISO standards with full Ferrero formatting.

  • 7-day grid — Day, Afternoon & Night shifts per day
  • ✓ Mandatory (green) & ! If-Needed (yellow) tasks
  • Section grouping by building & area
  • Weekday vs Weekend frequency columns
  • Weekly hours estimate column
📊 Export

ISO-Format Excel Export

Download .xlsx files matching the exact Ferrero ISO sheet layout — powered by ExcelJS.

  • Multi-tab — one worksheet per building/area
  • Header block: Area, Code, Revision, Approved By
  • D/A/N sub-columns per day with borders
  • Dark navy section headers, formatted task rows
  • Comments rows at bottom (optional)
🔤 Accountability

Worker Initials in Exports

Each cell shows WHO completed the task — not just a checkmark, but the worker's initials.

  • Green fill = Completed — e.g., "JS" for John Smith
  • Orange fill = Cleaned if Needed
  • Deterministic getInitials() with full test suite
  • Fallback: "N/A" for missing, "??" for invalid input
⚙️ Export Options

Smart Export Configuration

Full control over what gets exported before the file is generated.

  • Week picker — select any Mon–Sun date range
  • Toggle: include or exclude comments section
  • Toggle: completed rows only or all rows
  • Selections persisted in localStorage

  • Tab chips for each building/area worksheet
  • Mini grid with initials-filled sample cells
  • Green / orange / blank legend
  • Confirm then download — no surprise exports

Export Workflow

Dashboard to ISO Sheet in 4 Steps

The complete export flow — from selecting your week to receiving the formatted Excel file.

1
Configure Export Options

Select week range (Mon–Sun), toggle comments rows on/off, choose to include incomplete rows or completed only. Settings remembered for next visit.

2
Preview Before Downloading

A preview modal shows tab chips per building and a mini grid with initials-filled cells — see exactly what the export looks like before generating the file.

3
Generate with Gold Spinner

Click "Download .xlsx" — gold loading spinner appears, buttons are disabled to prevent duplicates, ExcelJS builds the file in the background.

4
Receive ISO-Formatted Excel File

One tab per building, each cell shows worker initials (e.g., "JS") with green or orange fill — a full compliance record matching Ferrero's ISO sheet format exactly.
